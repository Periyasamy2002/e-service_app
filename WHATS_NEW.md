### Start Server
```bash
python manage.py runserver
```

### Visit These URLs
- **Home**: `http://127.0.0.1:8000/`
- **About**: `http://127.0.0.1:8000/about/`
- **Contact**: `http://127.0.0.1:8000/contact/`

### Login as Different Roles
1. **Admin** - Manage services, assign work
2. **Agent2** - Review and forward requests
3. **Agent1** - Process assigned work
4. **User** - Browse and apply for services

## 📞 **QUICK REFERENCE**

### CSS Variables
```css
var(--primary)      /* #2563eb - Blue */
var(--success)      /* #10b981 - Green */
var(--danger)       /* #ef4444 - Red */
var(--background)   /* #f8fafc - Light gray */
```

### Using Animations
```html
<div style="animation: fadeIn 0.5s ease-out;"></div>
<div style="animation: slideDown 0.6s ease-out;"></div>
<div style="animation: scaleIn 0.3s ease-out;"></div>
```

### Responsive Classes
```html
<!-- Grid responsive -->
<div class="grid grid-2">
  <card></card>
  <card></card>
</div>

<!-- Full responsive -->
@media (max-width: 768px) { /* Tablet */ }
@media (max-width: 480px) { /* Mobile */ }
```
